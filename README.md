
# 🏠 Housing Price Prediction - Linear Regression

This project uses Linear Regression to predict housing prices based on features such as crime rate, room count, and location metrics from the Boston Housing dataset.

## 📊 Dataset
- Source: [Boston Housing Dataset](https://raw.githubusercontent.com/selva86/datasets/master/BostonHousing.csv)
- Target: `medv` - Median value of owner-occupied homes

## 🔧 Tools Used
- Python, Pandas, NumPy
- Scikit-learn (Linear Regression, Evaluation Metrics)
- Seaborn & Matplotlib (Visualization)

## 📈 Output
- Mean Absolute Error
- Mean Squared Error
- R² Score
- Actual vs Predicted plot

## 🚀 How to Run

1. Download `house_price_predictor.py`
2. Make sure required libraries are installed:
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```
3. Run the file:
```bash
python house_price_predictor.py
```

## ✅ Result
You'll get a summary of model performance and a plot showing how close predicted prices are to actual values.
